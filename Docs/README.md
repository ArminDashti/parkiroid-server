## Quick summary

**Parkiroid Server** is the central backend for monitoring remote field devices — typically camera-equipped hardware deployed in the field. It collects camera snapshots and health data from devices, keeps that information for a limited time, and enables live video viewing when operators need real-time visibility.

The system is **self-hosted** (runs on your own infrastructure), is in **early active development** (started June 2026), and works alongside separate client applications that operators and devices use to connect to it.

Start with [Product Overview](./product-overview.md) for the full picture.
